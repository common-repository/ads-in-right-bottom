Add Image banner ads, Flash banner ads, Text link ads, Custom Ads code (Google adsense, Yahoo Publisher Network, Adbrite, Clicksor...) in right bottom.
== Description ==
This plugin could bring so many kind of ads to right bottom, with opacity effect. 
Good for user want to add advertising banner over content.
This plugin will be power when intergate with [Custom Ads Sidebar](http://wordpress.org/extend/plugins/custom-ads-sidebar/) plugin.

[Demo here!](http://www.liemmp.com/yordes-docs.html)

== Installation ==


1. Upload the ads-in-right-bottom directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Plugins/Ads link bottom to configure.
4. Copy code `<?php
 if(function_exists('adslink')) { adslink(); } 
 ?>` and paste to anywhere you want to display ads (recommend footer.php)
5. That's it. Every new post will now be automatically add ads image.
